## 项目模块划分
- 网关模块
  路由转发、登录校验
- 跑批模块
  管理定时任务、界面启停
- 会员模块
  会员、乘客、以购买的车票
- web模块
  会员相关界面
- 业务模块
  所有的车次数据、余票信息
- admin模块
  管理员相关界面
